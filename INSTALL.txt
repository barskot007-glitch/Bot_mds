ОБНОВЛЕНИЕ CRM «ЗВЕЗДОСЛАВ»

1. Откройте корень репозитория на GitHub.
2. Загрузите содержимое этой папки, сохраняя структуру путей.
3. Подтвердите замену существующих файлов и сделайте Commit changes в ветку main.
4. Дождитесь автоматического deployment Railway.
5. Переменные Railway и DATABASE_URL менять не требуется.
6. Миграции базы для этого обновления не требуются.

Проверка после запуска:
/admin → База пользователей
/admin → Рассылки
/admin → Мероприятия
/admin → Управление админами
/admin → Статистика
