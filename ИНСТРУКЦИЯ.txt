ОБНОВЛЕНИЕ АДМИН-ПАНЕЛИ «ЗВЕЗДОСЛАВ»

1. Откройте репозиторий бота на GitHub.
2. Нажмите Add file → Upload files.
3. Перетащите папки app и tests из этого архива в корень репозитория.
4. Нажмите Commit changes.
5. Дождитесь успешного deployment в Railway.

Переменные Railway и PostgreSQL менять не нужно.
Миграций базы в этом обновлении нет.

Изменения:
- в карточке рассылки убраны «Добавить кнопку», «Файлы и кнопки», CSV-выгрузки;
- обращения после ответа остаются до нажатия «Завершить разговор», затем полностью удаляются;
- журнал показывает последние 10 действий понятным языком;
- добавлена выгрузка полного журнала в XLSX;
- статистика сокращена до основных показателей;
- общий экспорт оставлен только в XLSX;
- поиск пользователя возвращает только текстовую карточку без кнопок;
- кнопка «Настройки» удалена из админ-меню.
