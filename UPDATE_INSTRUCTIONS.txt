ОБНОВЛЕНИЕ БОТА «ЗВЕЗДОСЛАВ»

1. Распакуйте архив.
2. В GitHub откройте корень репозитория бота.
3. Нажмите Add file -> Upload files.
4. Перетащите все папки и файлы из этого архива в корень репозитория.
5. Подтвердите замену существующих файлов и нажмите Commit changes.
6. Railway автоматически создаст новый deployment.
7. Дождитесь статуса Success/Active.

Во время deployment автоматически применится миграция Alembic:
- в таблицу users будут добавлены поля phone и email;
- существующие данные пользователей не удаляются;
- уже зарегистрированные пользователи без телефона/email получат запрос этих данных при следующем /start.

FAQ скрыт через переключатель:
app/keyboards/admin.py
SHOW_FAQ_IN_ADMIN_MENU = False

Чтобы вернуть FAQ:
SHOW_FAQ_IN_ADMIN_MENU = True

Расширенные пункты пользовательского меню скрыты через:
app/keyboards/user.py
SHOW_EXTENDED_MENU = False

Чтобы вернуть их:
SHOW_EXTENDED_MENU = True
